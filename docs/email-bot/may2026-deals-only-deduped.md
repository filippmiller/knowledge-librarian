## 1. Услуги и возможности (capability)

1. **Перечень услуг.** Перевод с нотариальным заверением и без; истребование повторных документов в различных органах; справка об отсутствии судимости (в т.ч. срочно); апостилирование; консульская легализация; нострификация образовательных документов.
2. **Апостиль на ЕГРЮЛ/ЕГРИП** — оформляется (Минюст), в т.ч. с переводом.
3. **Чужой перевод заверить отдельно нельзя** — документ переводится заново, расчёт по ставке перевода.
4. **Альтернативы нотариальному заверению:** (а) заверение штампами бюро (исходник+перевод+страница с подписью переводчика и печатью компании), (б) просто перевод в Word. `[💰 живой расчёт]`
5. **Если нотариус не заверяет несколько документов одним** — предлагается заверение штампами компании (всё одним документом). `[💰 живой расчёт]`
6. **Два апостиля подряд** на документе — заверить можно, но ситуация нестандартная, предварительно уточняется у нотариусов.
7. **Электронный апостиль** — заверение возможно после проверки самого документа.
8. **Вычитка носителем / редактура профильным специалистом** — доступны как отдельные варианты. `[💰 живой расчёт]`
9. **Перевод только выделенного фрагмента** документа — возможен.
10. **Дублирование имён другим алфавитом** (напр. латиницей в скобках) — по просьбе.
11. **Нотариальный дубликат документа** с переводом и заверением — единой услугой.
12. **Письмо о квалификации переводчика** — может быть приложено.
13. **Перевод заверительной надписи** (подписи переводчика/нотариуса) на иностранный язык — по требованию.

## 2. Требования к документам (requirement)

14. **Оригинал обязателен для нотариального заверения** — перевод подшивается к оригиналу (можно привезти заранее или в день выдачи; в день выдачи ожидание ~40 мин).
15. **Скан:** чистый полный разворот без посторонних объектов; при плохом качестве часть текста уйдёт как «неразборчиво», просят переснять.
16. **Печать/подписи на документе** обязательны для нотариального заверения; без печати нотариус может отказать.
17. **Апостиль на ЕГРЮЛ:** оригинал с живой печатью и подписью налоговой → апостиль на нот. копию (дешевле); если только электронная с ЭЦП → нужна электронная нотариальная тождественность (дороже).
18. **Нострификация:** требуется перевод документа о предыдущем образовании с нот. заверением; номер диплома и приложения должны совпадать, иначе возврат на корректировку (срок ~5 дней) и отказ.
19. **Юрлицо при получении** для подписания закрывающих — печать организации либо оригинал доверенности.

## 3. Процесс (process)

20. **Перевод иностранного документа с апостилем + апостиль РФ:** нот. копия с оригинала (с апостилем) → перевод → нот. заверение → подача в Минюст на апостиль.
21. **Срок апостиля** — до 30 рабочих дней; при отсутствии образца подписи в Минюсте делается запрос в выдавший орган (дольше).
22. **Перевод подшивается к оригиналу** (к относящемуся документу/экземпляру); к ч/б или цветной распечатке — по согласованию.
23. **Согласование перед заверением:** перевод направляется клиенту на проверку (ФИО, имена, термины, серии/номера выделяются); заверяется только после подтверждения.
24. **ФИО латиницей:** по загранпаспорту или по правилам транслитерации; при смене фамилии — по согласованию; для единообразия берут написание из прошлых заказов. Желаемое написание лучше указать при оформлении.
25. **Документы из Молдовы** — перевод указывается с румынского (офиц. язык; зависит от года/страны выдачи).
26. **Нот. заверение** выполняется в день получения оригиналов; ожидание зависит от загрузки нотариуса (до нескольких часов). Заверение штампами компании при сшитии — ~15–30 мин.
27. **Тождественность** направлять по месту требования как можно быстрее — ключи доступа нотариуса могут меняться.
28. **Если апостиль поставили ПОСЛЕ перевода** — нужен новый перевод (перевод должен включать апостиль).
29. **Если нот. копия невозможна** — перевод подшивается к обычной ксерокопии.
30. **Уведомление о готовности** — звонок клиенту, как только нотариус вернул заверенный перевод.

## 4. Цены — ПОЛИТИКА (pricing_policy) `[💰 суммы из живого расчёта]`

31. **Расчёт перевода** — по условным страницам: 1 у.с. = 1800 знаков с пробелами (Word). У.с. ≠ физ. страница. Окончательно — по знакам готового перевода (предварительная сумма может слегка измениться, обычно в бóльшую сторону).
32. **Нотариальное заверение** — отдельно за каждый документ (итог = тариф × число документов).
33. **Срочность** — наценка; есть тарифная сетка по сроку (на следующий день / +1 / +2 раб. дня — чем дольше, тем дешевле). Заверение от срочности не зависит.
34. **Без нот. заверения** — дешевле на фикс. сумму заверения за документ; перевод в Word без ДНЗ; цена самого перевода почти не меняется.
35. **Сканирование готовых переводов** — отдельная платная услуга, поштучно за страницу (если не оговорено заранее).
36. **Частичная оплата невозможна** — счёт на полную предварительную стоимость (она предоплатная, итог по знакам).

## 5. Оплата и запуск заказа (policy)

37. **Запуск только после подтверждения оплаты** (скрин/чек/платёжка). Юрлицу обязательно прислать платёжное поручение. Оплату вносить **в рабочие часы 9:00–19:00**; при оплате после — запуск позже. В назначении платежа по карте писать ничего не нужно.
38. **Предоплата** для запуска, остаток — по готовности. Чтобы сохранить названную цену/срок — оплатить в тот же день до указанного менеджером времени.
39. **Акты** направляются на подписание ПОСЛЕ выдачи готовых документов; готовое по безналу/ЭДО высылается после доплаты.
40. **Возврат средств** (напр. двойная оплата): рукописное заявление в свободной форме (шапка: директору ООО «Аврора» от ФИО), скан/фото → рассмотрение руководством → возврат тем же способом.

## 6. Закрывающие документы / ЭДО (policy)

41. **ЭДО:** счёт, акт, УПД — по ЭДО (Контур.Диадок / СБИС / Диадок): клиент шлёт приглашение/идентификатор, компания принимает и направляет на подпись. Без ЭДО — бумажно или скан; УПД подписывается при получении в офисе (с печатью и доверенностью). По запросу в счёт добавляют номер заказа.
